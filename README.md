# ETL Pipeline for Air Quality Data

## Overview
This project demonstrates a simple ETL pipeline that extracts raw CSV data,
transforms it (cleaning, normalization), and loads it into a structured format.

## Features
- Data cleaning (missing values, duplicates)
- Feature engineering
- Output to CSV
- Simple visualization-ready output

## Tech Stack
Python, Pandas, NumPy

## How to Run
pip install -r requirements.txt
python main.py

## Output
Processed dataset saved in outputs/processed_data.csv
